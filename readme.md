# Josa-Complete
한국어 조사를 완성시켜주는 자바스크립트 라이브러리.

# How to use
## ES6 Modules
```javascript
import 'josa-complete'
console.log("우리집".을를) //console => "우리집을"
console.log("지갑".은는) //console => "지갑은"
```
## CommonJS
```javascript
requier("josa-complete")
console.log("나무".을를) //console => "나무를"
console.log("아몬드".은는) //console => "아몬드는"
```
